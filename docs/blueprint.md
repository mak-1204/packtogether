# **App Name**: PackTogether

## Core Features:

- Collaborative Itinerary Planning: Allow friends to collectively create, view, and modify trip itineraries in a shared digital space.
- Group Expense Tracking: Enable users to track planned versus actual trip expenses for all members and the overall budget.
- Trip Creation & Invitation: Facilitate creating new trip plans and inviting friends to join the planning session via a simple shareable link.
- AI-Assisted Trip Recommendation Tool: A generative AI tool that analyzes submitted trip suggestions against predefined group preferences to recommend the best option with an explicit rationale.
- Dynamic Landing Page: A responsive, single-page marketing site composed of a hero section with a dynamic image slideshow, features grid, a 'how-it-works' timeline, and a clear footer.

## Style Guidelines:

- The application will primarily use a dark theme by default to convey a sophisticated and modern feel. The primary brand color, a vibrant teal, will be used for interactive elements and highlights (HSL 175, 83%, 31%, equivalent to #0D9085).
- A deeply desaturated teal background color (HSL 175, 15%, 7%, equivalent to #0F1716) will provide a calm, modern backdrop suitable for a dark theme.
- A warm yellow/gold will serve as an accent color (HSL 38, 92%, 50%, equivalent to #F7A90A) to provide contrast and draw attention to important information or calls to action.
- All text throughout the application will use the 'Inter' sans-serif font for its modern, clean, and highly legible appearance across all screen sizes.
- The application will exclusively use 'lucide-react' for all icons, maintaining a consistent, clear, and scalable icon set across the UI, including the Compass for the logo, Map, Banknote, Bot, PartyPopper, Link2, ListChecks, and Plane icons for various content sections.
- A full-screen height hero section sets a bold visual tone upon entry, complemented by a logo positioned in the top-left for clear brand presence. Main content is centered for impact.
- Features will be presented in a clean three-column grid, making information easily digestible, while the 'How It Works' section utilizes a distinct four-step visual timeline for guided user understanding.
- Subtle animations enhance the user experience, including a dynamic slideshow with a gentle Ken Burns zoom-and-pan effect for background images in the hero section, transitioning every 5 seconds.
